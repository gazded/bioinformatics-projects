## Soil Metagenome Analysis

**Анализ влияния загрязнения почвы керосином на бактериальные сообщества с использованием данных 16S rRNA и QIIME2.** 
**Цель:** оценить влияние слабого (k5) и сильного (k25) загрязнения по сравнению с контролем (k0) на состав сообществ и их разнообразие.  

**Автор:** Слепов Константин Олегович | Использовал **SRA проект:** [PRJNA786393](https://www.ncbi.nlm.nih.gov/sra)

## Single-Cell RNA-seq Analysis of PBMC

**Анализ транскриптомов единичных клеток мононуклеарных клеток периферической крови (PBMC) с использованием Scanpy.**

**Цель:** Построение UMAP-репрезентации данных, идентификация клеточных кластеров и аннотация основных иммунных популяций с использованием канонических маркерных генов.

**Автор:** Слепов Константин Олегович  
**Данные:** 10X Genomics CellRanger output (H068_sample_filtered_feature_bc_matrix)  
**Технологии:** Scanpy, Python, Single-cell RNA-seq

**Основные этапы анализа:**
- Загрузка данных 10X Genomics → Контроль качества
- Нормализация → Отбор вариабельных генов  
- PCA → UMAP визуализация
- Кластеризация Leiden → Аннотация по маркерам

**Обнаруженные клеточные популяции:** Naive T-клетки, Memory T-клетки, NK-клетки, B-клетки, моноциты, дендритные клетки, тромбоциты, эритроидные клетки.
