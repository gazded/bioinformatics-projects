## Проект: Soil Metagenome Analysis

Анализ влияния загрязнения почвы керосином на бактериальные сообщества с использованием данных 16S rRNA и QIIME2.  
Цель: оценить влияние слабого (k5) и сильного (k25) загрязнения по сравнению с контролем (k0) на состав сообществ и их разнообразие.  

**Автор:** Слепов Константин Олегович | Использовал **SRA проект:** [PRJNA786393](https://www.ncbi.nlm.nih.gov/sra)
