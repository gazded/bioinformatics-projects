# Анализ влияния загрязнения почвы керосином на состав бактериального сообщества

В рамках проекта исследуется влияние различного уровня загрязнения почвы керосином на состав бактериального сообщества в дерново-подзолистой почве.

Почвенные образцы были помещены в закрытые ёмкости и обработаны разным количеством керосина. Отбор образцов проводился на 3-й, 90-й, 180-й и 360-й день эксперимента. Из каждого образца выделяли тотальное ДНК с использованием кита DNeasy PowerSoil.

Амплификация проводилась для вариабельных регионов V4–V5 гена 16S rRNA с использованием праймеров:
515F 5′–GTGCCAGCMGCCGCGGTAA–3′
907R 5′–CCGTCAATTCCTTTGAGTTT–3′

Секвенирование выполнялось на платформе **Illumina MiSeq** в парноконцевом режиме с длиной чтения 250 bp. Из-за снижения качества на концах R2-прочтений для анализа использовались только прямые (R1) чтения.

Данные взяты из NCBI SRA из проекта [PRJNA786393](https://www.ncbi.nlm.nih.gov/sra/?term=PRJNA786393).

---

## Цель проекта

Оценить влияние:
- слабого загрязнения (k5 — 5 г керосина/кг почвы),
- сильного загрязнения (k25 — 25 г/кг),
на бактериальные сообщества в сравнении с контролем (k0 — без керосина),  
на протяжении четырёх временных точек (3, 90, 180 и 360 день).
